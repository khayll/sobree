Sobree is an open-source, print-view-first OOXML document engine for building Word-compatible editing workflows inside your own app. Use it when a document-heavy product needs real `.docx` import, paginated editing, programmable mutations, and export without routing everything through an HTML-first editor.

## Hello world

```ts
import { createSobree } from "@sobree/core";
import { keyboard } from "@sobree/keyboard";
import { blockTools } from "@sobree/block-tools";
import "@sobree/core/tokens.css";

const editor = createSobree("#editor", {
  content: "# Hello\n\nStart typing.",
  plugins: [keyboard(), blockTools()],
});

editor.on("change", ({ doc }) => console.log(doc.body.length, "blocks"));
```

`createSobree()` mounts the viewport and the paginated paper stack;
the toolbar, keyboard shortcuts, and zoom dock arrive as opt-in
plugins you install separately. See the [Quick start](/quick-start/)
for the full options.
